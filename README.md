# mathwebsity

This website was created to help my friends and classmates study and stay organized throughout the school year. It includes important class notes, review materials, and summaries of key topics covered in class. The site also features quizzes and practice questions to help users prepare for tests and improve their understanding. A dedicated Regents review section provides extra practice and resources for exam preparation. The project was designed to make studying more accessible, interactive, and effective for students.
